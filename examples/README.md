# Examples Directory

Place your test images here:

- `content.jpg` - Your content image (the image whose content you want to preserve)
- `style.jpg` - Your style image (the image whose artistic style you want to apply)

These images will be used by the `example.py` script to demonstrate the neural style transfer.

You can use any JPEG or PNG images. For best results:
- Use high-quality images
- Content images work best when they have clear subjects
- Style images can be paintings, artistic photographs, or any image with a distinctive visual style
